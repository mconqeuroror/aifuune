# Session: 2026-07-09 — aifune

## User request

Push the AI Fuňe project to https://github.com/mconqeuroror/aifuune.git after downloading all static assets locally so they can later be uploaded to Vercel Blob and served from CDN.

## What changed

- **`src/lib/asset-base.ts`** — `ASSET_BASE` + `assetUrl()` helper driven by `VITE_ASSET_BASE`.
- **`src/lib/content.ts`**, **`src/lib/olympics-content.ts`**, **`src/lib/emoji-icons.ts`**, **`src/components/sections/HeroAndProof.tsx`** — all image/emoji paths use `assetUrl()`.
- **`public/fonts/`** — self-hosted DM Sans + Space Grotesk woff2 (21 files).
- **`src/fonts.css`** — local `@font-face` rules; imported from `src/index.css`.
- **`index.html`** — removed Google Fonts CDN links.
- **`scripts/download-fonts.py`** — downloads font files from Google Fonts CSS.
- **`scripts/generate-asset-manifest.mjs`** — writes `public/assets.manifest.json` (128 static files).
- **`vercel.json`** — SPA rewrite for React Router.
- **`.env.example`** — documents `VITE_ASSET_BASE` for blob CDN.
- **`vite.config.ts`** — rewrites font URLs when `VITE_ASSET_BASE` is set.
- **`README.md`** — dev + blob deployment instructions.
- **`package.json`** — `assets:manifest` and `fonts:download` scripts.
- **`.gitignore`** — temp extraction logs, `tsconfig.tsbuildinfo`.

## Why

User wants the repo on GitHub with zero runtime CDN dependencies, ready to upload `public/` to blob storage and flip one env var on Vercel.

## Gotchas

- Fonts have multiple woff2 files per weight (unicode-range subsets); filenames include range slugs.
- Duplicate `inline-*` images in `public/images/` are legacy extraction copies; canonical paths use `all-b64-*`.
- `VITE_ASSET_BASE` must not have a trailing slash.
- Until blob is configured, leave `VITE_ASSET_BASE` empty — assets serve from `/public`.
